# Chapter 5: Core Vocabulary by Category (Wortschatz)

This chapter contains essential thematic vocabulary lists. To help you learn nouns effectively, they are listed with their definite articles (**der** for masculine, **die** for feminine, **das** for neuter) and their plural endings (e.g., *die Hand, "-e* means the plural is *die Hände*).

---

## 1. Greetings and Basics (Begrüßungen und Grundlagen)

| German | English |
| :--- | :--- |
| **Hallo!** | Hello! |
| **Guten Morgen!** | Good morning! |
| **Guten Tag!** | Good day / Hello! |
| **Guten Abend!** | Good evening! |
| **Gute Nacht!** | Good night! |
| **Auf Wiedersehen!** | Goodbye! (Formal) |
| **Tschüss!** | Bye! (Informal) |
| **Bitte** | Please / You're welcome |
| **Danke** | Thank you |
| **Ja / Nein** | Yes / No |
| **Entschuldigung** | Excuse me / Sorry |
